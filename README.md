# Hello World

Projek Hello World sederhana - Simple Hello World project

## 📋 Persyaratan / Requirements

- Browser modern dengan support HTML5 dan CSS3
- Text editor atau IDE (VS Code, Sublime Text, dll)

## 🚀 Memulai / Getting Started

### 1. Buka File / Open File

Cukup buka file `index.html` di browser Anda, atau gunakan Live Server di VS Code:
- Klik kanan pada `index.html` 
- Pilih "Open with Live Server"

Atau:
```bash
# Jika menggunakan Python (tersedia di folder proyek)
python -m http.server 8000
```

Kemudian buka `http://localhost:8000` di browser.
